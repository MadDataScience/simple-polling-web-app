Simple Web Polling App

new user login

Welcome to my simple web polling app. This app will allow you to create simple polls (yes/no questions only), adminster them, and see results. If this is your first time here, click on new user to create your account.

Once you create your account, login to administer your polls.

Once some polls have been created, you will be able to see them below. Feel free to fill them out:

    {{range .Polls}}
  • {{.Title}}
    • {{end}}

Once polls have been filled out, login and you will be able to see results on the edit page for the poll in question.

The source code for this app can be found at github.com/MadDataScience/simple-polling-web-app
